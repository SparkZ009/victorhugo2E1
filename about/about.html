    Sobre o Trabalho:
  • Este trabalho foi proposto pelo professor Paulo Henrique Pereira Silva, cujo mesmo leciona a disciplina de Artes Técnica no Colégio Cotemig, e pela professora Luciene de Souza, que leciona a matéria de química, a ideia do trabalho foi de que, fizéssemos uma tabela periódica que linkasse cada átomo em uma respectiva página para que nestas páginas tivessem algumas características importantes de cada átomo.
    Em minha opinião, este trabalho foi ótimo para que pudessemos aprender ainda mais sobre a programação de sites, no caso o dev web.
